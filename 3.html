데이터 타입

Javascript에서 사용되고 있는 데이터 타입에 대하여 정리한다.

데이터 타입

값을 기준으로 형태를 분류한 것으로 자바스크립트 프로그램에서 사용할 수 있는 6개 타입과 자바 스크립트 컴파일러 개발자를 위한 7개 스펙상의 타입을 정의한다.

  1. 프로그램에서 사용할 수 있는 타입

    2. 실체가 있으며 데이터를 저장, 추출, 변경, 삭제할수 있음

    • Number
    • String
    • Undefined
    • Null
    • Boolean
    • Object
  2. 컴파일러 개발자를 위한 스펙상의 타입

    3. 자바스크립트 내부 처리를 위한 타입으로 자바스크립트 프로그램에서 사용할 수 없다. Lexical Environment는 자바스크립트를 깊게 이해하기 위해필요, 24장에서 공부

    • List
    • Completion
    • Property Descriptor

    • Lexical Record

정수와 실수

자바스크립트는 정수와 실수를 구분하지 않는다.

  1. 정수

    2. -9007199254740992 ~ 9007199254740990 사이의 값을 말하며, 12e3(=12*1000)과 같은 지수를 사용할수 있음

        ex code)
	console.log(123);
	console.log(0129);
	console.log(+0);
	console.log(-0);
	console.log(12e3);


  2. 부동 소수점

    3. 123.45와 같은 소수점을 포함한 값을 의미, 자바스크립트는 정수와 실수를 구분하지 않으며 모든 정수, 실수를 64비트 부동 소수점으로 계산, 즉 123를 123.0으로 계산한다. 부동소수점은 1.23e2와 같은 지수(e)를 사용하여 소수 이하의 수를 표현할수 있다. 여기서 지수란? 1.23*(10*10)를 말한다.

        ex code)
	console.log(0.12);
	console.log(0.12e3);
	console.log(12e-3);
	console.log(12.e-3);



Infinit, NaN

  1. Infinit

    2. 무한대 값을 나타내며 양수 무한대와 음수 무한대가 있습니다. +Infinit는 양수무한대 -Infinit는 음수무한대를 나타냄. 계산값이 양수무한대면 Infinit를 반환하며, 음수무한대면 -Infinit를 반납한다.

        ex code)
	console.log(1.7976931348623157e308);
	console.log(1.7976931348623157e309);
	console.log(-1.7976931348623157e308);
	console.log(-1.7976931348623157e309);


  2. NaN(Not-a-Number)

    3. 값이 숫자가 아니거나 계산할 수 없는 값을 나타낸다. 0을 0으로 나누면 NaN을 반환합니다.

        ex code)
	console.log(0/0);



진수

정수값을 10진수, 8진수, 16진수로 작성할수 있다. 8진수와 16진수로 음수 값을 나타낼수 있지만 유동소수점과 지누(e)를 나타낼 수 없다. 일반적으로 10진수를 사용, 16진수, 8진수는 특수한 경우에만 사용한다.

  1. 16진수

    2. 16진수에서 0~9의 수는 숫자로 사용하며, 10~15의 수는 a~f를 사용한다. a~f는 대소문자를 구분하지 않고 사용 가능. 앞에 0x 혹은 0X를 사용한다.

        ex code)
	console.log(0Xf);
	console.log(0XFD);
	console.log(0X7e3)


  2. 8진수

    3. 8진수는 0~7 숫자로 표현하여 작성하며, 앞에 0를 붙여서 사용한다.

        ex code)
	console.log(01);


typeof 연산자

typeof의 다음에 작성한 데이터(값)의 타입을 반환하는 연산자

typeof연산자 결과
typeof 대상 데이터 타입 반환 값
Undefined 'undefined'
Null 'object'
Boolean 'boolean'
Number 'number'
String 'string'
{설명-1} 참조 Object
{설명-2} 참조 function
{설명-3} 참조 브라우저 개발사에서 정의 
    ex code)
	console.log(typeof 123);
	console.log(typeof 12e3);
	console.log(typeof 0/0);
	console.log(typeof value);


{설명-1}

네이티브 오브젝트(Native Object)이면서 호출할 함수가 설정되지 않았을 때 Object를 반환, 상세한 설명은 9장 Object에서 설명

{설명-2}

네이티브 오브젝트이면서 호출할 함수가 설정되었을때 function을 반환, 상세한 설명은 16장 Function 오브젝트에서 설명

{설명-3}

브라우저 개발사에서 반환 값을 결정하며, undefined, boolean, number, string이 아니어야 함. 예를 들어 node, element, document와 같은 값이 반환됨.


따옴표 사용

문자 또는 문자열은 작은 따옴표(' '), 큰따옴표(" ")를 사용하며, 작은 따옴표와 큰 따옴표를 중복으로 사용할 수 있다.

+로 문자열과 문자열을 연결할 수 있다.

    ex code)
	var sports = '축구 : ';
	player = '11명';
	console.log(sports + player);


숫자와 문자열 더하기

숫자와 문자열을 더하면(+) 숫자에 문자열을 연결하여 반환, 숫자가 앞/뒤에 있을때의 결과가 상이함으로 아래의 예제를 참조한다.

    ex code)
	var sports = '축구 : ';
	player = '11';
	console.log(sports + player);
	console.log('A' + 1 + 2);
	console.log(1 + 2 + 'A');
	console.log(1 + 2 + 'A' + 3 + 4);


오브젝트 타입

자바스크립트에서 오브젝트의 의미가 크며 비중도 매우 높다.

  1. 오브젝트 형태

    2. {book : 'JavaScript', sport: 'soccer'}

    오브젝트 작성 방법

    1. 중괄호{}를 작성합니다. {}는 오브젝트를 나타냄
    2. 오브젝트 안에 {이름:값} 형태와 같이 이름, 콜론(:), 값을 하나의 단위로 작성
    3. {book : 'JavaScript'}에서 book이 이름, JavaScript가 값
    4. 콤마로 구분하여 "이름:값"을 여러개로 작성 가능
    5. 이름과 값이 하나도 없는 오브젝트는 빈 오브젝트

  2. 프로퍼티와 속성

    3. {sport: {soccer : '11명'}}

    이름과 값을 프로퍼티(Property)라고 하며, 오브젝트의 기본 구성단위 임.

    값에는 자바스크립트에서 제공하는 모든 데이터 타입이 작성될 수 있다.

    속성(Attribute)은 프로퍼티의 상태를 나타내거나 값의 속성을 나타내며 {sport: {soccer : '11명'}} 오브젝트에서 {soccer : '11명'}은 sport의 값이며, 속성은 오브젝트임.


블리언 타입

불리언(Boolean)은 true와 false를 나타내며 숫자로 표현하면 1(=true), 0(=false)로 표현한다. 아래의 예제를 보고 기능 확인

    ex code)
	var value = true;
	console.log(value);
	console.log(typeof value);
	console.log(true + 2);
	console.log('A' + false);


undefined 타입

undefined 값도 데이터 타입이며, var sports;와 같이 변수 선언하면 자바스크립트에서는 sports를 undefined를 할당합니다. sports변수값의 초기값이 undefined라는 것이다. 비록 변수 선언만 했지만 undefined값을 갖고 있다.

    ex code)
	var sports;
	console.log(sports);
	console.log(undefined);
	console.log(typeof undefined);
	console.log(undefined + 1);
	console.log(2 + undefined);
	console.log('A' + undefined);
	sports ? console.log('값 있음') : console.log('값 없음');


null 타입

null은 값이며 데이터 타입이다. undefined는 변수만 선언한 것이고 null은 변수 선언 후 null로 값을 치환한 것이다. 프로그램에서 undefined와 null을 정확히 구분해서 사용하여야 프로그램을 정확하게 개발할수 있다.

    ex code)
	console.log(null);
	console.log(typeof null);
	console.log(null + 1);
	console.log(2 + null);
	console.log('A' + null);
